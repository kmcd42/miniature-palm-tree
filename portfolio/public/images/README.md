# Images Directory

Place your images in this directory following these naming conventions:

## Required Images

### Homepage
- `hero-image.jpg` - Main hero image for homepage
- `project-1.jpg`, `project-2.jpg`, `project-3.jpg` - Featured projects

### About Page
- `about-photo.jpg` - Personal photo for about page

### Portfolio
Place project images in `projects/` subdirectory:
- `projects/bolton-hotel.jpg`
- `projects/karawhiua.jpg`
- `projects/silverstripe.jpg`
- `projects/wearableart.jpg`
- `projects/doctor-who.jpg`
- `projects/nice-little-palaces.jpg`
- `projects/project-zero.jpg`

### Photography
Place photography images in `photography/` subdirectory organized by collection:
- `photography/abstract-cover.jpg`
- `photography/abstract-1.jpg` through `photography/abstract-8.jpg`
- `photography/urban-cover.jpg`
- `photography/urban-1.jpg` through `photography/urban-8.jpg`
- `photography/documentary-cover.jpg`
- `photography/documentary-1.jpg` through `photography/documentary-8.jpg`

### Contact
- `contact-1.jpg`, `contact-2.jpg` - Decorative images for contact page

## Image Guidelines
- Use high-quality images (at least 1920px wide for hero images)
- Optimize images before uploading (use tools like ImageOptim, TinyPNG)
- Prefer JPEG for photos, PNG for graphics with transparency
- Next.js will automatically optimize images at runtime
